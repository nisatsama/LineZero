# 🚀 ZeroLine – AI-Powered Smart Productivity Assistant

> **Your Last-Minute Life Saver** 💡
> Plan smarter. Focus better. Never miss a deadline again.

---

## 📌 Overview

ZeroLine is an **AI-powered productivity assistant** built to eliminate last-minute stress by helping users intelligently manage tasks, schedules, goals, and deadlines.

Powered by **Google Gemini AI**, ZeroLine goes beyond traditional to-do apps by automatically prioritizing tasks, generating optimized schedules, providing personalized productivity insights, and proactively reminding users before important deadlines.

Built with modern web technologies and deployed on **Google Cloud**, ZeroLine delivers a fast, scalable, and responsive productivity experience across desktop and mobile devices.

---

## ✨ Features

### 🤖 AI-Powered Productivity

* Intelligent task prioritization using the **Eisenhower Matrix**
* AI-generated daily schedules optimized according to workload
* Personalized productivity recommendations
* Automatic task breakdown into smaller actionable subtasks
* AI Productivity Coach for planning, studying, and workflow optimization

---

### 📋 Smart Task Management

* Smart Planner Dashboard
* Create, edit, organize, and track tasks
* AI-powered project categorization

  * Learning
  * Work
  * Personal
  * Fitness
  * Others
* Smart Notes workspace
* Goal & Habit Tracker
* Streak monitoring and milestone tracking

---

### ⏰ Deadline & Reminder System

* Context-aware deadline reminders
* Voice alerts:

  * 30 minutes before
  * 15 minutes before
  * 5 minutes before
* Calendar integration
* Export schedules as `.ics` files compatible with Google Calendar and Apple Calendar

* Built-in Pomodoro Focus Timer

* Task completion analytics
* Focus statistics

## ⚙ Installation

### Clone the repository

```bash
git clone https://github.com/nisatsama/LineZero.git
```

### Navigate into the project

```bash
cd LineZero
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
